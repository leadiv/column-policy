Column Policy Example

This page highlights how the column policy mixin along with media queries can be used to dynamiclly change grid layout types. When the browser size is greater then 1120 pixels then the page will show up as a three column layout with the left and right column fixed at a 320px width. When the browser is sized down below 1080 pixels then the layout changes to a two column grid layout with both columns fluid. Once the browser width drops below 720 pixels then the page becomes a single column layout.
